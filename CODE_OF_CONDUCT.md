# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1, as its code of conduct.

In short: be respectful and constructive. We want a welcoming, professional community
for everyone contributing to orvacon, regardless of background or experience level.

## Scope

This applies to all project spaces — the repository, issues, pull requests, discussions —
and to public spaces when an individual is representing the project.

## Reporting

If you experience or witness unacceptable behavior, report it privately to
**conduct@orvacon.com**. Reports are handled confidentially. Maintainers are responsible
for clarifying standards and may take appropriate corrective action.

For the full text, including the enforcement guidelines, see the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
